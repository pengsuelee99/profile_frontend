# 🛠️ ຄູ່ມືການຈັດການ Portfolio ເອງ (Handbook)

ເຈົ້າສາມາດເພີ່ມຜົນງານ, ປ່ຽນຮູບ, ຫຼື ໃສ່ Link ໃໝ່ໆໄດ້ງ່າຍໆຕາມ 3 ຂັ້ນຕອນນີ້:**

---

### 1. ການຈັດການຮູບພາບ (Images) 📸
- **ບ່ອນເກັບ**: ເອົາຮູບໄປວາງໄວ້ໃນ `portfolio-frontend/public/projects/`.
- **ຊື່**: ໃຫ້ຕັ້ງຊື່ເປັນພາສາອັງກິດ (ເຊັ່ນ: `web1.png`, `app2.jpg`).

### 2. ການອັບເດດຂໍ້ມູນ (Data) ✍️
- **ໄຟລ໌**: ເປີດໄຟລ໌ `portfolio-backend/src/projects.json`.
- **ການແກ້ໄຂ**:
    - **ປ່ຽນຮູບ**: ແກ້ໄຂແຖວ `"image": "/projects/ຊື່ຮູບຂອງເຈົ້າ.png"`.
    - **ໃສ່ Link**: ແກ້ໄຂແຖວ `"link": "https://facebook.com/..."` (ຫຼື Link GitHub ຂອງເຈົ້າ).
    - **ເພີ່ມໂປຣເຈັກໃໝ່**: Copy ກຸ່ມ `{ ... }` ມາຕໍ່ທ້າຍແລ້ວປ່ຽນຂໍ້ມູນທາງໃນ.

### 3. ການເບິ່ງຜົນ (Sync) 🔄
- **Auto-Refresh**: ລະບົບຈະດຶງຂໍ້ມູນໃໝ່ໃຫ້ເອງທຸກໆ 30 ວິນາທີ! ເຈົ້າພຽງແຕ່ Save ໄຟລ໌ JSON ແລ້ວຖ້ານິດໜຶ່ງ ຮູບ ແລະ ຂໍ້ມູນກໍຈະປ່ຽນເອງເລີຍ.

---

**ໝາຍເຫດ**: ຫ້າມລືມເຄື່ອງໝາຍຈຸດ `,` ຢູ່ທ້າຍແຕ່ລະ Block ແລະ ໃຫ້ກວດເບິ່ງນາມສະກຸນຮູບ (.png / .jpg) ໃຫ້ຖືກຕ້ອງສະເໝີເດີ້! 🚀🐾
